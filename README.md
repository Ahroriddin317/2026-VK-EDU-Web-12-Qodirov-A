# AskMe — Web-сервис вопросов и ответов (ДЗ №2)

Учебный проект в рамках курса **VK EDU "Web-технологии"**.  
Это вторая стадия разработки: перенос статической верстки на фреймворк **Django**, настройка маршрутизации, логики отображения и контейнеризации.

**Автор:** Кодиров Ахрориддин  
**Группа:** WEB-12

---

## 🛠 Технологический стек
* **Backend:** Python 3.12 + Django 5.x
* **Frontend:** Bootstrap 5.3 + Django Templates
* **DevOps:** Docker, Docker Compose

---

## 📂 Структура проекта
Проект приведен к стандартной структуре Django-приложения:
- `app/` — логика приложения (views, models, urls).
- `application/` — ядро проекта и настройки (settings.py).
- `static/` — общие статические файлы (CSS, JS, изображения).
- `templates/` — HTML-шаблоны с использованием наследования (`base.html`).
- `uploads/` — директория для пользовательского контента.
- `Dockerfile` & `docker-compose.yml` — конфигурация для быстрого развертывания.

## Что реализовано в ДЗ №2

Django Integration: Весь статический контент перенесен в шаблоны Django.

Named URLs: Все переходы между страницами реализованы через именованные маршруты {% url %}.

Верстка форм: Созданы «ровные» страницы для Login, Signup, Ask Question и Profile Settings.

Pagination: Реализована универсальная функция пагинации для всех списков (главная, горячие вопросы).

Base Template: Использовано наследование шаблонов для исключения дублирования кода навигации и футера.

Docker: Проект полностью контейнеризирован для стабильной проверки на любой машине.

## Реализованные страницы :
index/ — Главная страница со списком новых вопросов.

hot/ — Горячие вопросы .

question/ — Страница одного вопроса с ответами.

login/ — Авторизация пользователя.

signup/ — Регистрация нового аккаунта.

ask/ — Форма создания нового вопроса.

settings/ — Настройки профиля.
