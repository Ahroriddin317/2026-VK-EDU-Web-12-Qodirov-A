# AskMe — Статическая верстка (ДЗ №1)

Учебный проект в рамках курса VK EDU "Web-технологии".  
**Автор:** Коидров Ахрориддин  
**Группа:** WEB-12

## Описание
В данном проекте реализована статическая верстка сервиса вопросов и ответов (аналог StackOverflow). Верстка выполнена с использованием **Bootstrap 5.3**.

## Структура проекта
- `public/` — основная директория с HTML-файлами.
- `public/static/` — локальные статические файлы (CSS, JS, изображения).

## Как запустить проект
Так как это статическая верстка, установка дополнительных зависимостей не требуется.

1. Клонируйте репозиторий:
   ```bash
   git clone [https://github.com/Ahroriddin317/2026-VK-EDU-Web-12-Qodirov-A.git](https://github.com/Ahroriddin317/2026-VK-EDU-Web-12-Qodirov-A.git)

2. Перейдите в папку проекта:

   ```Bash
   cd ваш-репозиторий/public

3. Откройте файл index.html в любом современном браузере (Chrome, Firefox, Safari).
